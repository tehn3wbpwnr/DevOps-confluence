DevOps / Pipeline & Deployment : Set Credentials in Jenkins

Created by Jonathan Bezeau on Apr 09, 2025

We need to navigate to the “folder” on Jenkins but not the actual Pipeline, it should look like this.

We should end up here

We want to click the “Add Credentials” button in the top right.

Again we want it to be a “Secret Text” Type, and the id must be “bitbucket-access-token” then you add your copied Bitbucket access token into the secret. Then you can click create.

Now you can head to your actual pipeline and click “Build Now” This will run the pipeline for the first time and populate some of the parameters. This run is intended to fail which is okay.

From here the pipeline should be setup, when a PR is made it should trigger the pipeline to run, running through the assigned JenkinsFile
