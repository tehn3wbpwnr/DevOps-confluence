Introduction

Through this page, we will walk you through creating a pipeline on Jenkins. This will introduce you to some of the advanced settings within a Bitbucket repository, as well as how to navigate the Jenkins settings. For the project you can use one of your onboarding projects from the first 2 weeks, or any Unity repository will work. We will be using a relatively empty JenkinsFile so we donâ€™t need to worry about building or testing. You will need a Jenkins Account, please contact a team member to ensure you have the correct access.
