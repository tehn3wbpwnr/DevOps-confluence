DevOps / Pipeline & Deployment : Connect to our portal server

Created by Jonathan Bezeau, last modified on Apr 14, 2025

Our entry point server is at vconestoga.com and we can connect to it very simply:

Run command prompt, terminal, powershell, git bash, or your choice of command line.

At the command line enter ssh -i .ssh/vconkey.pem vconadmin@vconestoga.com

exit to disconnect from vconestoga
