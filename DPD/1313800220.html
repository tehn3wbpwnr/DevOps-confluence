Other Consideration and References

  • Property-based Testing

    • Definition

      • Property-based Testing is different from Example-based Testing, which is typically used in Unit and Integration Testing. In Example-based Testing, you assume specific inputs and expected outcomes for each case. In contrast, Property-based Testing defines what types of inputs are allowed and the expected properties or invariants that the code should always satisfy. **The testing framework then automatically generates all possible inputs to verify those properties across a wide range of scenarios.
