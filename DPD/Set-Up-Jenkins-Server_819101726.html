DevOps / Pipeline & Deployment : Set Up Jenkins Server

Created by Jonathan Bezeau, last modified on Oct 05, 2023

  • Connect to build VM using Remote Desktop Protocol

  • Download and install Java

    • prefer JDK17, Jenkins will complain about JDK21

  • Download and install Jenkins

    • Choose JDK (can update in config files later if version changes)

    • Configure to communicate on port 80

    • didn’t config an admin account so Jenkins will generate a password for user “admin” in install directory

      • stash credentials in Bitwarden

  • Use start bar to bring up Services

    • Start Jenkins service

  • Browse http://<IP address> to bring up Jenkins dashboard

    • login as admin

  • Install plugins

    • All Bitbucket plugins

      • except for any Bitbucket Server

    • Dashboard View

    • Generic Webhook Trigger Plugin

    • NUnit plugin

    • Perforce (reserved for future use)

    • Pipeline

Follow up with steps to create a project
