2. BrightSpace(eConestoga)

: Based on the BrightSpace documentation, the Tool Provider should set the 'keyset URL' to communicate with BrightSpace.

However, BrightSpace uses different ‘keyset endpoint’ compared to Moodle like the table below.

Platform

keyset endpoint

Moodle

'/keys'

BrightSpace

'/jwks'

So, making sure whether the LTI service has a proper keyset endpoint you want to communicate with.
