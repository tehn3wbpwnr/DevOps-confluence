About Moodle

: Moodle (Modular Object-Oriented Dynamic Learning Environment) is an open source learning management system (LMS) and a platform that supports online learning.

It is currently being used as a test environment related to LTI Integration in VARLab.

1. How to Create An Account

: First, you need an admin account to create a new account.

1.1 Go to “Site administration” >> “Users“ >> “Add a new user“

1.2 Fill all the user information and create a new user

2. How to Register An External Tool

2.1 Go to “Site administration” >> “Plugins“ >> “Manage tools“

2.2 Enter the “tool_url/register” and Click “Add LTI Advantage“ Button

2.3 Click “Activate“ button for pending external tool (now good to go)

3. How to Create A Course

3.1 Go to “Site home“ and Turn on the editing mode

3.2 Scroll to the bottom and click the "Add new course" button

3.3 Fill in all the blanks for course information and create a new course

4. How to Create A Content

4.1 Go to the course you want to add content to and Turn on the “editing mode“

4.2 Click “Add an activity or resource“ on the topic you want

4.3 Select “External tool“

4.4 Select external tool at “Preconfigured tool” selection

4.5 Click “Select“ button >> Select a dlx project you want to add at course >> Submit

4.6 Click “Save and return to course” or “Save and display“

5. How to Assign Learners in A Certain Course

: Students can’t see the course before they are assigned to the certain course. So, if we want specific students to have access to a course, you must assign those students to the course.

5.1 Go to the course you want to manage >> “Participants“ >> “Enrol users“

5.2 Search the user and Add to the course

5.3 Check enrolled user
