DevOps / Pipeline & Deployment : Create a Repository

Created by Jonathan Bezeau, last modified on Apr 09, 2025

Sign into Bitbucket, click Create Repository and fill in the details. We can’t really get anywhere without creating a repository on the origin system.

Clone the repository to your computer

If using git in a terminal window you’ll need to clone the repository to your local environment before checking out a branch. This will creates a folder structure in your files with subfolders and metadata information that lets git know that this workspace is tracked and by whom so that later git commands like branch and commit have a context to work with.
