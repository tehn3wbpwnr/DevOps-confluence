DevOps / Pipeline & Deployment : Delete and Clean up Pipeline

Created by Chris Park on Apr 09, 2025

This documentation outlines the process for cleaning up pipelines of projects that have already been released.

Delete project folder in the Jenkins website

1. Select the project folder you want to delete

2. Click 'Delete Folder'

3. Click 'Yes' when delete folder message pop ups

4. Confirm the folder deletion at the Dashboard

Delete project folder in the Jenkins VM

cd ~/var/lib/jenkins/workspace

3. Verify the existence of the project folder

4. Delete the project folder from the working directory
